# INFO MASTER 📚  
정보관리 기술사 준비를 위한 학습/기출 관리 웹 서비스

이 프로젝트는 정보관리 기술사(ITPE) 공부를 위해  
기출문제 관리, 답안 작성, 노트 정리 기능을 제공하는  
**React + Spring Boot 3 + PostgreSQL 기반 웹 서비스**입니다.
